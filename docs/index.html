项目介绍

SwapFace 是一个基于 Python、Flask、OpenCV 和 InsightFace 的人脸替换/换脸项目。
支持实时视频流处理，适合学习、演示和研究人脸识别与图像处理技术。

功能亮点

高效换脸

基于 InsightFace 和 ONNX，支持高质量人脸替换。

实时处理

支持摄像头视频流实时换脸，低延迟体验。

易于扩展

代码结构清晰，便于二次开发和集成更多功能。

开源友好

适合学习、演示和科研用途，欢迎贡献和交流。
